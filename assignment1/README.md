# Nodejs

Node.js is a free, open-source, cross-platform JavaScript runtime environment that allows developers to build server-side applications, command line tools, and other types of applications using JavaScript. <br>
Node.js utilizes the V8 JavaScript engine, the same engine used in Google Chrome, making it fast and powerful 

## Steps to install Node

- update package index : sudo apt update
- install node.js : sudo apt install nodejs
- verify the installation : node -v 
- installing node package manager : sudo apt install npm 
- verify installation : npm -v

## command to run nodejs code 

- node index.js
